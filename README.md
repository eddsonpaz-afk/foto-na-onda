# Foto na Onda

Aplicativo da experiência **Foto na Onda da Produtividade**.

## Rodar no computador

```bash
npm install
npm run dev
```

## Publicar no Vercel

1. Envie os arquivos para o GitHub.
2. No Vercel, importe o repositório.
3. Framework: Vite.
4. Build Command: `npm run build`
5. Output Directory: `dist`

## Trocar a moldura

Substitua:

`public/moldura-oficial.png`

por uma imagem PNG 1080x1920 com o mesmo nome.

## Logos oficiais

Nesta base, as logos aparecem como texto provisório. Substitua o componente `Brand` por imagens oficiais em PNG ou SVG antes da publicação final.
