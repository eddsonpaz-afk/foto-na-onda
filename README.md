# Foto na Onda V2

Versão integrada com:

- Waves Plus
- ExpoConstruir
- câmera do celular
- moldura 1080 × 1920
- Google Sheets
- Google Drive
- compartilhamento nativo/WhatsApp
- download da foto

## Publicação

1. Substitua os arquivos do repositório por estes.
2. Faça commit no GitHub.
3. O Vercel publicará automaticamente.
4. Framework: Next.js.
5. Não configure diretório de saída.

A URL do Google Apps Script já está configurada como fallback no código.
