# Foto na Onda V2.1 — enquadramento corrigido

Correções desta versão:

- câmera traseira abre primeiro;
- botão para alternar entre câmera traseira e frontal;
- foto pode ser reposicionada arrastando na tela;
- controle de zoom;
- botão para recentralizar;
- captura final respeita exatamente o enquadramento visto;
- área inferior da moldura agora é opaca e não deixa a foto vazar;
- abertura central ampliada para enquadramento mais natural.

## Publicação

Substitua os arquivos do repositório pelos arquivos deste pacote e faça o commit.
O Vercel publicará a atualização automaticamente.
