# Foto na Onda — Next.js

Projeto preparado para implantação direta no Vercel.

## Estrutura

- `app/page.jsx`: telas e funcionamento
- `app/globals.css`: visual
- `public/moldura-oficial.png`: moldura aplicada à câmera

## Vercel

- Framework: Next.js
- Build command: `npm run build`
- Output directory: deixe automático

## Trocar a moldura

Substitua `public/moldura-oficial.png` por um PNG 1080 × 1920 mantendo o mesmo nome.
